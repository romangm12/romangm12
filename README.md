# Roman David Gonzalez Mora

Biologist working at the intersection of molecular biology, genomics, and computational tools.

I use code to make laboratory workflows easier to calculate, document, visualize, and reproduce. My current work includes NGS preparation tools, interactive scientific figures, thesis visualizations, and small web applications for communicating data.

## Featured Work

### Equimolar Calculation for NGS Sequencing

Python calculator for preparing equimolar pools of DNA fragments or amplicons before next-generation sequencing.

**Focus:** molecular biology, NGS preparation, Python, scientific calculation.

[View repository](https://github.com/romangm12/Equimolar_Calculation_NGS_sequencing)

### Interactive Figures for Virus-Host Modeling

Interactive figures based on evolutionary game theory and reactive payoff matrices for virus-host interaction models.

**Focus:** scientific visualization, interactive figures, modeling, evolutionary theory.

[View repository](https://github.com/romangm12/Interactive-figures-modeling-of-virus-virus-)
